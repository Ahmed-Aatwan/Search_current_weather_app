# 🌤️ Weather App (Flutter)

A modern weather application built with Flutter that provides real-time weather updates with a dynamic UI that changes based on weather conditions.

## ✨ Features

• 🔍 Search weather by city name

• 🌡️ Real-time temperature updates

• 🌤️ Dynamic UI based on weather conditions

• 📊 Max / Min temperature display

• 🔮 Forecast weather information

• 🎨 Glassmorphism-inspired UI design

• ⚡ Clean state management using BLoC (Cubit)

## 🧠 State Management

This project uses:

**Flutter BLoC (Cubit pattern)**

## 🏗️ Architecture

• Cubit for state handling

• Clean separation of UI and logic

• Reusable custom widgets (GlassCard, InfoItem, etc.)

## 📦 Tech Stack

• Flutter

• Dart

• flutter_bloc

• cached_network_image

• HTTP / API integration
