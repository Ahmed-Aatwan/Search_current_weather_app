🌤️ Weather App (Flutter)

A modern weather application built with Flutter that provides real-time weather information with a clean, dynamic UI that changes based on weather conditions.

✨ Features
🌍 Search weather by city name
🌡️ Real-time temperature display
🌤️ Dynamic UI based on weather conditions
📊 Max / Min temperature display
🔮 Forecast weather condition
🎨 Modern glassmorphism-inspired UI
⚡ Smooth state management using BLoC (Cubit)
🧠 State Management

This project uses:

flutter_bloc (Cubit pattern)
🏗️ Architecture
Cubit for state handling
Separation of UI and logic
Reusable widgets (GlassCard, InfoItem, etc.)
📦 Packages Used
flutter_bloc
cached_network_image
http (or dio if used)
intl (if used for formatting)
